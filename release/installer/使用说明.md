# Vue Element Plus Admin 安装使用说明

## 安装包位置
- 安装包：`release\installer\VueElementPlusAdmin-Setup.exe`

## 安装前准备
- Windows x64，管理员权限
- 如提示缺少运行库，请先安装 **Microsoft Visual C++ 2015-2022 x64 运行库**
- 默认端口：Web = 3002，MariaDB = 3307

## 安装步骤（内置数据库模板版）
1. 右键安装包选择“以管理员身份运行”。
2. 选择安装目录（建议默认）。
3. “数据库配置”页面保持默认即可。
4. **“初始化数据库和用户”不要勾选**（本安装包已内置表结构模板）。
5. 完成安装后，桌面快捷方式会打开系统。

## 访问方式
- 浏览器打开：`http://localhost:3002`
- 若无法访问：
  - 打开“服务”，确认 `VueElementPlusAdmin` 与 `VueElementPlusAdminDB` 均为“正在运行”。

## 默认账号
- 用户名：`admin`
- 密码：`123456`

## 数据与配置位置
- 数据目录：`{安装目录}\backend\mariadb-data`
- 数据库配置：`{安装目录}\backend\config.json`
- 初始化日志：`{安装目录}\backend\install-db.log`

## 端口冲突处理
- Web 端口 3002 被占用：修改 `{安装目录}\bin\start-backend.cmd` 中的 `PORT=3002`，并同步修改防火墙规则（如需）。
- 数据库端口冲突：安装时会自动切换空闲端口并写入 `config.json`。

## 升级/备份
- 升级：新安装包选择**同一安装目录**即可保留数据。
- 备份：停止服务后复制 `{安装目录}\backend\mariadb-data`。
