各模块使用方法与关键流程说明
登录与权限 首页概览 居民管理 人民调解 待办提醒 特殊人群 机构管理 系统设置 常见问题

登录与权限

  • 使用分配的账号密码登录系统。
  • 不同角色拥有不同的菜单与按钮权限。
  • 如无法访问某模块，请联系管理员分配权限。

首页概览

  • 展示居民、低收入、残疾人、通知等统计数据。
  • 事项进度：统计“任务”型待办的进度分布。
  • 纠纷处理趋势：近 6 个月调解档案数量趋势。

居民管理

居民查询

  • 支持按姓名、身份证号、组别等条件筛选。
  • 可导出当前筛选结果为 Excel。

字段说明（常用）

字段 说明
姓名 居民姓名，必填。
身份证号码 18 位身份证号，必填，用于年龄/性别自动解析。
与户主关系 “本人”代表户主，其它成员应填写与户主关系。
民族 来自字典管理中的民族选项。
组别 村组信息，用于生成户编号与统计。
户籍详细地址 用于户主匹配与家庭成员归户。

新增/编辑居民

  • 身份证号、民族等信息用于自动计算年龄与校验。
  • 与户主关系为“本人”的记录视为户主。

业务流程

居民列表 → 新增/编辑 → 保存 → 查询/导出

批量导入

  • 流程：上传文件 → 字段映射 → 数据预览 → 导入完成。
  • 必填字段：姓名、身份证号码。
  • 户主识别依据：与户主关系 = “本人”。

导入字段建议

Excel 表头 建议映射字段
姓名 name
身份证号码 / 身份证号 id_card
与户主关系 relationship_to_head
组别 village_group
户籍详细地址 / 地址 Home_address

人民调解

调解申请书

  • 录入纠纷类型、发生日期、申请人/被申请人信息。
  • 可从居民库快速选择人员并自动填充身份证、民族等。

字段说明（申请书）

字段 说明
纠纷类型 纠纷分类，用于统计与筛选。
发生日期 纠纷实际发生日期。
申请人/被申请人 支持多人，身份证/民族等会自动回填。

调解记录

  • 记录调解过程与结果，并可上传现场图片。
  • 若“是否达成协议”选择“是”，则不再允许新增调解记录。

业务流程

创建档案 → 填写申请书 → 新增调解记录 → 达成协议 → 生成协议书
截图示例：调解记录时间轴、调解记录表单（请替换为实际截图）

调解协议书

  • 只有在调解记录中选择“达成协议”后才可填写。
  • 支持生成并打印协议书 PDF。

待办提醒

  • 支持“任务 / 事件提醒 / 系统”等类型。
  • 仅“任务”类型显示进度条。
  • 支持标记已读、批量操作与关键字搜索。

字段说明

字段 说明
类型 任务/事件提醒/系统；仅任务显示进度。
进度 0-100，用于任务进度。
状态 pending/read 等状态。

特殊人群

低收入管理

  • 登记低收入人员信息，支持查看与编辑。
  • 可导出列表或按条件筛选。

业务流程

查询居民 → 建立低收入档案 → 编辑/维护 → 导出

残疾人管理

  • 登记残疾人员信息及证件信息。
  • 支持导出与状态维护。

机构管理

  • 维护现任及历届两委班子、团妇、村民小组长信息与职务数据。
  • 支持新增、编辑与导出。

字段说明（常用）

字段 说明
姓名 / 职务 用于展示与排序。
联系方式 用于通知或联系。

通知配置

  • 配置生日提醒等通知参数。

常见问题

导入提示“格式错误”

  • 请确认已映射 “姓名” 与 “身份证号码”。
  • Excel 某行缺少姓名/身份证号会被判为错误。

接口 404/400

  • 确认后端已启动且接口路径为 /api 前缀。
  • 遇到 400 错误可查看弹窗的“错误详情”。
