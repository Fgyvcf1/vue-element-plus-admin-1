村委会管理系统（Windows）快速上手（10 分钟）

目标：在 Windows 电脑上完成最简部署并能访问系统
默认路径：D:\deploy_bundle

----------------------------------------
一、先准备（必须）

1) 安装 Node.js（LTS）
- 下载：nodejs.org
- 安装路径建议：D:\nodejs
- 把 D:\nodejs 加到系统 PATH
- 验证：node -v

2) 安装 MariaDB（10.6+）
- 下载：mariadb.org
- 端口：3306
- 设置 root 密码并记住
- 验证：mysql -u root -p

3) 解压 Nginx（Windows 版）
- 下载：nginx.org
- 解压到：D:\nginx

4) 准备部署包
- 目录结构：
  D:\deploy_bundle\frontend\
  D:\deploy_bundle\backend\

----------------------------------------
二、按顺序执行（最短路径）

第 1 步：初始化数据库（自动脚本）
- 进入：D:\deploy_bundle\backend
- 运行：setup-local-mariadb.bat
- 按提示输入 root 密码、数据库名、业务用户名
- 记下脚本输出的“数据库密码”（后面要用）

第 2 步：启动后端服务
- 仍在 D:\deploy_bundle\backend
- 运行：start.bat  或  start-backend.bat
- 按提示输入 DB_PASSWORD（不回显）
- 看到提示：后端服务运行在 http://localhost:3002

第 3 步：配置 Nginx 反向代理
- 打开 D:\nginx\conf\nginx.conf
- 在 server 里配置：

server {
  listen 80;
  server_name localhost;
  root D:/deploy_bundle/frontend;
  index index.html;

  client_max_body_size 50m;

  location / {
    try_files $uri $uri/ /index.html;
  }

  location /api/ {
    proxy_pass http://127.0.0.1:3002;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
  }

  location /uploads/ {
    proxy_pass http://127.0.0.1:3002;
  }

  location /archives/ {
    proxy_pass http://127.0.0.1:3002;
  }
}

- 重载 Nginx：D:\nginx\nginx.exe -s reload

第 4 步：访问验证
- 浏览器访问：http://localhost/
- 能看到登录页即成功

----------------------------------------
三、遇到问题怎么办（最常见）

1) 80 端口冲突
- 把 listen 80 改成 8080
- 用 http://localhost:8080 访问

2) 登录/接口报错
- 确认后端是否在 3002 端口运行
- 重新运行 start.bat 并输入正确 DB_PASSWORD

3) 模块缺失
- 进入 backend 运行：npm rebuild

—— 完 ——
